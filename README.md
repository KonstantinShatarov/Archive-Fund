# Архивный фонд

Приложение предназначено для работы с архивом образовательной организации. 
В нем реализованы функции управления пользователями, включая добавление, редактирование, удаление, блокировку и разблокировку учетных записей. Также предусмотрены возможности создания резервных копий базы данных и восстановления системы в случае ошибок.
Приложение обеспечивает хранение информации о личных делах студентов, их специальностях, видах выполняемых работ и самих выполненных работах. 
При первом запуске приложения система автоматически создает базу данных, если она отсутствует на сервере. Для настройки приложения под конкретный сервер необходимо внести изменения в конфигурационный файл .
Если пользователь соглашается с автоматическим созданием базы данных, система также генерирует стартовую учетную запись, предназначенную для первоначальной настройки приложения, включая создание и управление учетными записями пользователей. 
С благодарностью принимаю предложения по улучшению и замечания относительно представленного материала.

[![image.png](https://i.postimg.cc/4xDwPZDm/image.png)](https://postimg.cc/K15nvXv2)

Программа «Архивный фонд» является инновационным программным решением, предназначенным для использования в образовательных учреждениях. Ее основная цель заключается в обеспечении эффективного электронного учета личных дел студентов, учебных специальностей, типов выполняемых работ и достигнутых результатов. 
Функциональные возможности программы включают: 
- управление записями: возможность добавления, редактирования и удаления информации о личных делах студентов, выполненных ими работах, учебных специальностях и типах работ. 
- фильтрация данных: возможность фильтрации информации по истечении срока хранения, а также по фамилии, имени и отчеству студентов (личные дела и работы). 
- экспорт данных: функция экспорта данных в формат PDF с указанием номера полки и стеллажа. 
- администрирование: наличие панели администрирования, доступной только для администратора, которая предоставляет возможности для добавления, редактирования и удаления пользователей, блокировки и разблокировки учетных записей, создания резервных копий базы данных и восстановления базы данных в случае необходимости. 
Программный продукт разработан на языке программирования C# с использованием интегрированной среды разработки Visual Studio. Интерфейс программы отличается удобством и интуитивно понятным дизайном, что обеспечивает эффективное взаимодействие с пользователем и эффективное управление данными.
